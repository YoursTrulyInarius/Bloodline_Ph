## 📌 Project Overview

**BloodLine Ph** is a web-based information system prototype designed to connect blood donors with patients in urgent need across the Philippines. It was developed as a Final Project for **Contemporary World** under the theme *"Tech for Change: IT Solutions for SDGs."*

## ❓ The Three Core Questions

### 1. What is the problem?

In the Philippines, finding an emergency blood donor is still largely done through **informal social media posts** — scattered Facebook statuses, group chats, and word-of-mouth appeals. This process is:

- **Slow** — posts get buried under other content within hours
- **Disorganized** — no centralized system to filter by blood type or location
- **Unreliable** — contact information is often outdated or unverified
- **Inaccessible** — people in provincial areas have even less access to donor networks

Every minute counts in a blood emergency. The lack of a dedicated, organized platform costs lives.

---

### 2. What SDG is involved?

**SDG 3 — Good Health and Well-being**

> *"Ensure healthy lives and promote well-being for all at all ages."*

Specifically, this project addresses:
- **Target 3.8** — Achieve universal health coverage, including access to quality essential health-care services
- **Target 3.d** — Strengthen the capacity of all countries for health risk reduction and management of health emergencies

Blood donation is a critical component of emergency healthcare. A reliable digital system for blood matching directly supports the goal of accessible, quality health services for all Filipinos — regardless of their location or social media reach.

---

### 3. How can IT help solve this problem?

Information Technology can replace the chaotic, informal system with a **structured, real-time digital platform** that:

| IT Feature | How It Helps |
|---|---|
| **Centralized Web Platform** | One place for all blood requests — no more scattered Facebook posts |
| **Real-time Data (localStorage + Storage Events)** | Requests appear instantly across all open sessions without page refresh |
| **Search & Filter (Find Match)** | Patients can filter donors by exact **blood type** and **city** in seconds |
| **Donor Registry** | Willing donors can pre-register so they can be found before an emergency happens |
| **Blood Bank Directory** | Lists verified blood banks with operating hours and contact numbers across the Philippines |
| **Mobile-Responsive Design** | Accessible on smartphones, which most Filipinos use as their primary internet device |

In a real-world deployment, this prototype could be scaled with a **cloud database (e.g., Firebase)**, **SMS notifications**, and **geolocation features** to create a fully functioning national blood donor network.

## 🚀 How to Run the Prototype

1. Download or clone the project folder.
2. Open `index.html` in any modern web browser (Chrome, Edge, Firefox).
3. No internet connection or server setup required.
4. To reset all data, open the browser console (F12) and type: `localStorage.clear()` then refresh.
